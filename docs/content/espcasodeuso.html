Caso de uso

Descrição do Caso de Uso

O FocaVest é um sistema online que auxilia vestibulandos a se prepararem para os vestibulares. A plataforma oferece diversas funcionalidades, como:

Criação de agendas personalizadas: Os usuários podem adicionar as datas dos vestibulares de seu interesse e acompanhar o cronograma de estudos.

Guias de estudo personalizados: O sistema oferece materiais de estudo adaptados ao perfil e às necessidades de cada usuário.

Ferramentas de acompanhamento: Os usuários podem acompanhar seu desempenho em simulados e exercícios, além de receber feedback personalizado.

Comunidade: A plataforma oferece um espaço para que os usuários possam tirar dúvidas, trocar experiências e se conectar com outros estudantes.

Atores

Pré-vestibulando Administrador Professor/Tutor (opcional)

Precondições

O usuário possui uma conta ativa no sistema: O usuário deve ter se cadastrado e criado um perfil no FocaVest.

O vestibular desejado está cadastrado no sistema: O administrador do sistema deve ter previamente cadastrado o vestibular em questão, incluindo informações como nome da instituição, data, cidade, etc.

O usuário está autenticado no sistema: O usuário deve estar logado em sua conta para realizar a ação.

A agenda do usuário não estar cheia: A agenda do usuário deve ter capacidade para adicionar mais eventos (vestibulares).

O usuário possui permissão para adicionar eventos à agenda: Essa pré-condição pode ser relevante caso existam diferentes tipos de usuários com diferentes níveis de acesso.

Pós-Condições

O vestibular selecionado é adicionado à agenda do usuário: A data do vestibular, o nome da instituição e outras informações relevantes são incluídas na agenda do usuário, na data e horário especificados.

A agenda do usuário é atualizada: A lista de eventos na agenda do usuário é atualizada para refletir a adição do novo vestibular.

O sistema gera uma notificação para o usuário: O usuário recebe uma notificação (por e-mail, aplicativo, etc.) confirmando a adição do vestibular à agenda.

O sistema registra a ação em um log: A ação de adicionar um vestibular à agenda é registrada em um log de atividades para fins de auditoria e análise.

Fluxo de Eventos

Fluxo Básico (Principal)

Adicionar um Vestibular à Agenda (Pré-Vestibulando)

PASSOS:

  1. O usuário acessa a seção "Minha Agenda". O usuário navega até a área da plataforma destinada à gestão de sua agenda pessoal.

  2. O usuário clica no botão "Adicionar Vestibular". O usuário localiza e interage com o elemento visual que inicia o processo de adição de um novo vestibular.

  3. O sistema apresenta uma lista ou campo de busca para o vestibular. O sistema exibe uma interface que permite ao usuário pesquisar ou selecionar o vestibular desejado a partir de uma lista pré-definida.

  4. O usuário seleciona o vestibular desejado. Após localizar o vestibular, o usuário confirma sua escolha, indicando ao sistema qual vestibular será adicionado à agenda.

Fluxos Alternativos

Usuário Adiciona um Vestibular Duplicado (Pré-Vestibulando) PASSOS:

  1. O usuário acessa a seção "Minha Agenda" e clica em "Adicionar Vestibular". (Mesmos passos 1 e 2 do fluxo principal)

  2. O usuário pesquisa ou seleciona o vestibular. O usuário encontra o vestibular desejado e o seleciona.

  3. O sistema verifica se o vestibular já está na agenda do usuário. O sistema consulta a agenda do usuário para verificar se o vestibular selecionado já está cadastrado.

  4. O sistema exibe uma mensagem de alerta. O sistema informa ao usuário que o vestibular já está cadastrado em sua agenda.

  5. O usuário pode: a. Editar o vestibular existente: O sistema direciona o usuário para a tela de edição do vestibular, onde ele pode alterar as informações, se necessário. b. Cancelar a operação: O usuário pode desistir de adicionar o vestibular.

O usuário tenta adicionar um vestibular que já ocorreu (Pré-Vestibulando) PASSOS:

  1. O usuário acessa a seção "Minha Agenda" e clica em "Adicionar Vestibular". (Mesmos passos 1 e 2 do fluxo principal)

  2. O usuário pesquisa ou seleciona o vestibular. O usuário encontra o vestibular desejado e o seleciona.

  3. O sistema verifica a data do vestibular. O sistema compara a data do vestibular selecionado com a data atual.

  4. O sistema exibe uma mensagem de alerta. O sistema informa ao usuário que a data do vestibular já passou.

  5. O usuário pode: a. Corrigir a data: O usuário pode voltar e corrigir a data do vestibular. b. Cancelar a operação: O usuário pode desistir de adicionar o vestibular.

Administrador remove vestibular do sistema (Administrador) PASSOS:

  1. O usuário acessa a seção "Minha Agenda" e clica em "Adicionar Vestibular". (Mesmos passos 1 e 2 do fluxo principal)

  2. O usuário pesquisa ou seleciona o vestibular. O usuário encontra o vestibular desejado e o seleciona.

  3. O sistema verifica se o vestibular está cadastrado. O sistema consulta o banco de dados para verificar se o vestibular existe.

  4. O sistema verifica se o vestibular foi removido. O sistema verifica se o vestibular foi removido por um administrador após a consulta inicial.

  5. O sistema exibe uma mensagem de erro. O sistema informa ao usuário que o vestibular não está mais disponível.

  6. O usuário pode: a. Realizar uma nova pesquisa: O usuário pode tentar encontrar outro vestibular. b. Contactar o suporte: O usuário pode entrar em contato com o suporte técnico para obter mais informações.
